# 易数乾坤

> 基于易经起卦仪式感与 AI 深度推演的数字彩票辅助娱乐工具

## 项目简介

"易数乾坤"将易经"起卦"的仪式感与 AI 的逻辑处理结合，通过"新中式极简"的审美视觉，为用户提供一种"玄学+科技"的双重心理慰藉，辅助生成彩票号码。

## 功能特性

- 🎯 **彩种选择**：支持双色球、大乐透等主流数字乐透
- 🎲 **起卦仪式**：六爻交互，铜钱翻转，生成专属卦象
- 🤖 **AI 推演**：结合卦象数据，智能生成推荐号码
- ✨ **精美动效**：水墨扩散、数字流动、签条下落等仪式感动画
- 📜 **灵签展示**：卦名印章、运势评语、推荐号码一体化呈现
- 📱 **多端适配**：微信小程序 + H5 网页端双支持

## 技术栈

- **前端框架**：原生 HTML/CSS/JavaScript
- **UI 样式**：Tailwind CSS
- **字体**：Noto Serif SC、ZCOOL XiaoWei
- **AI 接口**：DeepSeek-V4-Flash (SiliconFlow)

## 业务流程

1. **择法**：用户选择彩种玩法
2. **准入**：弹出合规风险声明（必须勾选）
3. **演卦**：交互 6 次，生成六爻状态
4. **化数**：AI 根据卦象推演号码
5. **得数**：渲染签条结果页，支持分享

## 核心功能

### 1. 择法首页
- 彩种切换器（双色球/大乐透）
- 太极按钮触发起卦
- 合规声明弹窗

### 2. 演卦仪式
- 三枚铜钱翻转动画
- 六爻自下而上堆叠
- 实时进度展示
- 计时器功能

### 3. AI 推演
- 卦象数据结构化传输
- DeepSeek AI 智能生成
- 本地降级算法备份
- 号码规则硬校验

### 4. 结果展示
- 宣纸签条下落动画
- 卦名印章展示
- 运势书法体展示
- 推荐号码球动画
